🔒 Тестирование безопасности системы администрирования

1. Тест хранения учетных данных

Проблема: Пароли хранятся в localStorage в открытом виде
⚠️ УЯЗВИМОСТЬ: Пароль администратора хранится в localStorage без шифрования. Любой скрипт на странице может получить доступ к паролю.

2. Тест защиты сессии

Проблема: Сессия проверяется только при загрузке страницы
⚠️ УЯЗВИМОСТЬ: После входа можно изменить sessionStorage вручную и получить доступ.

3. Тест защиты от брутфорса

Проблема: Нет ограничений на количество попыток входа
❌ КРИТИЧЕСКАЯ УЯЗВИМОСТЬ: Можно бесконечно пытаться подобрать пароль.

4. Тест таймаута сессии

Проблема: Сессия не истекает автоматически
⚠️ УЯЗВИМОСТЬ: Сессия остается активной до закрытия браузера.

5. Тест XSS уязвимостей

Проблема: Пользовательский ввод может содержать вредоносный код
⚠️ УЯЗВИМОСТЬ: Нет санитизации пользовательского ввода.

6. Тест прямого доступа к админ-панели

Проблема: Можно попытаться открыть админ-панель напрямую
