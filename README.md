# Веб-приложение детского сада "ТЕРЕМОК"

Веб-приложение для управления информацией о детском саде, разработанное с использованием XML, XSLT, HTML, CSS и JavaScript.

## Структура проекта

- `data.xml` - XML файл с данными о детском саде (группы, дети, расписание, мероприятия)
- `transform.xsl` - XSLT файл для преобразования XML в HTML
- `index.html` - Главная HTML страница приложения
- `styles.css` - Стили для оформления приложения
- `script.js` - JavaScript для навигации между разделами
- `loadXML.js` - JavaScript для загрузки и парсинга XML данных

## Возможности

1. **Информация о детском саде**
   - Название, адрес, контакты
   - Описание детского сада

2. **Группы**
   - Список всех групп с информацией о возрасте
   - Данные о воспитателях
   - Список детей в каждой группе с контактами родителей

3. **Расписание**
   - Ежедневное расписание занятий и активностей

4. **Мероприятия**
   - Календарь предстоящих мероприятий

## Запуск приложения

### Способ 1: Прямое открытие (работает без сервера!)

1. Просто откройте файл `index.html` в браузере (Chrome, Firefox, Edge)
2. Приложение автоматически использует встроенные данные, если не удастся загрузить `data.xml`
3. Все функции работают без необходимости запуска веб-сервера

### Способ 2: Прямое открытие XML с XSLT

1. Откройте файл `data.xml` в браузере (Chrome, Firefox, Edge)
2. Браузер автоматически применит XSLT трансформацию

### Способ 3: Локальный веб-сервер (для загрузки внешнего XML)

Если вы хотите редактировать `data.xml` и видеть изменения без перезагрузки страницы, используйте веб-сервер:

#### Использование Python:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### Использование Node.js:
```bash
npx http-server
```

#### Использование PHP:
```bash
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000/index.html`

## Особенности реализации

- **XML структура**: Данные организованы в иерархической XML структуре
- **XSLT трансформация**: Автоматическое преобразование XML в HTML
- **Адаптивный дизайн**: Приложение корректно отображается на всех устройствах
- **Интерактивность**: Переключение между разделами через вкладки
- **Современный UI**: Градиенты, анимации, карточки с эффектами наведения

## Редактирование данных

Для изменения данных откройте файл `data.xml` и отредактируйте нужные элементы:

- Добавление новой группы: скопируйте блок `<группа>...</группа>` и измените данные
- Добавление ребенка: добавьте новый блок `<ребенок>...</ребенок>` внутри `<дети>`
- Изменение расписания: отредактируйте элементы в разделе `<расписание>`
- Добавление мероприятия: добавьте новый блок `<мероприятие>...</мероприятие>` в `<мероприятия>`
